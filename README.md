## Sobre o projeto

Este repositório documenta minha jornada prática em **Análise de Dados com Python**, com foco em:

- Manipulação de dados com Pandas
- Operações numéricas com NumPy
- Limpeza e transformação de dados
- Análise exploratória (EDA)

---

## Evolução do aprendizado

Os exercícios seguem uma progressão:

1. NumPy (estrutura e operações)
2. Pandas básico
3. Criação e transformação de colunas
4. Análise com groupby e pivot_table
5. Manipulação com loc e iloc
6. Tratamento de dados (nulos)
7. Merge e análise entre tabelas
